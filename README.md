# Space-Adventure

これはピクセルのスタイルのワンページのWebサイトです。HTMLとCSSでWebの構成作成し、グラフィックデザインならasepriteを利用していきました。完全に細かくピクセルスタイルを作りたいのため、日本語の例文は８－ビットのフォントを利用していきました。


### コンセプト
ゲームテスターの求人のサイトです。売前のゲームをプレイして、バグ（不具合）や仕様通りに動かない点を発見し、開発チームに報告します。しかも、可能範囲の限りに、ゲームのグラフィック、ゲームプレイまたは改善可能の点があれば、報告することができます。

### プログラミング言語、デザインツール
* <ins>**HTML**</ins>:ウェブページの構造や内容を定義するためのマークアップ言語。<br/>
* <ins>**CSS**</ins>: ウェブサイトの詳細の部分をスタイリングのツール<br/>
* <ins>**Aseprite**</ins>: ドット絵（ピクセルアート）の作成とアニメーション制作に特化した人気のグラフィックエディタで、レトロゲーム風の素材制作やゲーム開発で広く使われています。<br/>
